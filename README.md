# Backup Monitor

Приложение для мониторинга резервных копий на сетевом хранилище (СХД).

## Требования

- Windows 10/11
- .NET 8.0 или выше
- Visual Studio 2022 или выше

## Структура проекта

- `Models/Service.cs` - модель сервиса бэкапов
- `Services/ConfigurationManager.cs` - управление конфигурацией (загрузка/сохранение)
- `Services/BackupChecker.cs` - логика проверки бэкапов
- `MainWindow.xaml` - главное окно приложения
- `Views/ServiceWindow.xaml` - окно настройки сервиса
- `Views/PeriodWindow.xaml` - окно выбора периода проверки
- `services.json` - файл конфигурации сервисов

## Использование

1. Запустите приложение
2. Добавьте сервисы через кнопку "Добавить сервис"
3. Для каждого сервиса укажите:
   - Название
   - UNC-путь к папке с бэкапами (например: `\\192.168.1.100\backups\db`)
   - Ключевые слова для поиска файлов (через запятую)
   - Регулярные выражения для извлечения даты из имени файла

4. Используйте кнопки:
   - "Проверить сегодня" - проверка бэкапов за текущий день
   - "Проверить период" - проверка бэкапов за указанный период

## Поддерживаемые форматы дат

- `yyyy_MM_dd` (например: 2026_01_06)
- `yyyy-MM-dd` (например: 2026-01-06)
- `ddMMyyyy` (например: 06012026)

## Конфигурация

Конфигурация хранится в файле `services.json` в корне приложения. Файл создаётся автоматически при первом добавлении сервиса.
